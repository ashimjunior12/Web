Functions

Why Functions:-
  1. Functions allow to reuse code.
  2. Same code can be used multiple times
JavaScript Function Syntax:-
Function in javascript are declared using function keyword followed by function name. After function name we use () parenthesis. Inside of it, there can be parameters or it can be empty.

function function_name(parameter1, parameter2){
    //block of code
}

Function Invocation:-
A function performs a certain task when it is invoked or called.

Function Return:- After using a function we expect the function to produce something. This is done using return statement. By using return statement, we are telling the function to give us something(output). The code written after return statement is not executed.

function sum(a, b){
   return a+b;        //returns the sum of two numbers and provides to add variable
   console.log("Hello")     //This will not execute
}
let add = sum(2,3);        //By "sum(2,3)", we are calling/invoking function.
console.log(`Sum: ${add}`);

Output:- Sum: 5

Parameters:-
Parameters are the variables that are passed inside a function. In above code, (a,b) are parameters.

Arguments:-
Arguments are the data which are passed into the method's parameter. In above code, (2,3) are arguments.

Default Parameter Values

function sum(a, b=10){
   return a+b;
}
console.log(`Sum: sum(5)`);

Output:- Sum: 15

In above code, if the value of b is not passed or is undefined, then it takes 10(default value).
