String Functions

  1. indexOf(); :-
    Indexing strats from 0.
    indexOf() function gives us the index of the substring. (First Occurence).

    var name = "Ram is a good boy and is tall";
    console.log(name.indexOf("is"));

    Output:- 4    //First occurence of "is" is at 4.

  2. lastIndexOf();
    lastIndexOf() gives the index of the string at last occurence.

    console.log(name.lastIndexOf("is"));

    Output:- 22    // Last occurence of "is" is at no 22.

  3. .slice(starting_point, ending_point);
    .slice() method is used to slice a string.

    console.log(name.slice(0, 3));

    Output:- Ram

    (In above case, 0 is included but 3 is excluded).

    .slice() takes negative values as well.

  4. .substr();
    Similar to the .slice() method, .substr() method is used to slice string.
    .substr() doesn't take negative value.

    console.log(name.substr(0,3));

    Output:- Ram

  5. .replace("to_be_replaced", "replaced_value");
    .replace() method is used to replace a substring with new value.

    console.log(name.replace("Ram", "Hari"));

    Output:- Hari is a good boy and is tall.

    Since strings are immutable data type, meaning the initial value of "name" is not affected by the replaced value. If we print out the name, we get the initial value of name.

    console.log(name);

    Output:- Ram is a good boy and is tall.

  6. .toUpperCase();
    As the name suggests, .toUppercase() method is used to convert a string to its uppercased format.

    console.log(name.toUpperCase());

    Output:- RAM IS A GOOD BOY AND IS TALL.

  7. .toLowerCase();
    Similar to .toUpperCase(); method, .toLowerCase() method is used to convert a string to its lowercased format.

    console.log(name.toLowerCase());

    Output:- ram is a good boy and is tall.

  8. .trim();
    .trim() method is used to remove whitespaces from the beginning and the end of a string.

    var spaces = "  Hello   World  !  "
    console.log(spaces.trim());

    Output:- Hello  World  !

    //The space is removed from start and end only,
    not in middle part of the string.

  9. .charAt(index);
    .charAt() method gives a character at the index assigned to it.

    console.log(name.charAt(1));

    Output:- a

  10. [index];
    variable name followed by [] with index inside of it gives the character at that index.

    console.log(name[1]);

    Output:- a

    In this case, we cannot use negetive index.
