Loops in JavaScript

Loops are used to repeat a block of code.
  1. For Loop:-
    Syntax of for loop:-

    for (initialization;condition;increment/decrement){
       //block of code
    }

    Example of for loop to display numbers:-

    for(let i=0; i<5; i++){
       console.log(i);
    }

    In above code, we first initialized the value of i which is equal to 0. After that we set the condition in which we specify that "i" should be less than 5. And after, the value of "i" is increased after each iteration until the condition is met.
    The output is as:-

    Output:-
    0
    1
    2
    3
    4

    Iteration Variable Condition Action
    1st i=0 true 0 is printed
    i becomes 1
    2nd i=1 true 1 is printed
    i becomes 2
    3rd i=2 true 2 is printed
    i becomes 3
    4th i=3 true 3 is printed
    i becomes 4
    5th i=4 true 4 is printed
    i becomes 5
    6th i=5 false loop is terminated


    Printing sum of 5 natural numbers using for loop:-

    let sum = 0;
    for(let i=1; i<=5; i++){
        sum += i;
    }
    console.log(`Sum: ${sum}`);

    Output:-
    Sum: 15



  2. While loop:-
    Syntax:-

    while(condition){
       //block of code
       //increment/decrement
    }

    Example of while loop to display numbers:-

    let i=0;      //Initializing the value of i
    while(i<5){
       console.log(i);
       i++;     //increasing the value of i
    }

    Output:-
    0
    1
    2
    3
    4



  3. Do-While loop
    Do-While loop executes a block of code for at least one time even if the condition is false.

    Syntax:-

    do{
       //block of code
    }
    while(condition);


    Example of do while loop to display 5 numbers:-

    let i = 0;         //Initializing
    do{
       console.log(i);
       i++;       //Incrementing value of i
    }
    while(i<5);       //condition checking


    Output:-
    0
    1
    2
    3
    4


  4. For-of Loop:-
    for-of loop is used to iterate over iterable objects such as arrays, strings, sets and maps.

    Example of looping an array using for loop

    let names = ["Ram", "Shyam", "Hari", 12, "Sita"];
    for (let i=0; i<names.length; i++){
        console.log(names[i]);
    }

    The for-of loop becomes handy when iterating over iterable objects because it simplifies the iteration process.

    Example of looping an array using for-of loop

    let names = ["Ram", "Shyam", "Hari", 12, "Sita"];
    for(name of names){
        console.log(name);
    }

    In both cases output is:-

    Output:-

    Ram
    Shyam
    Hari
    12
    Sita

  5. For-in loop:-
    for-in loop is used to iterate over properties of an object.

    Example:-

    let employee = {
       name: "Ram",
       salary: 200000,
       address: "kathmandu",
       age: 26
    };
    for (keys in employee){
       console.log(keys + ": " + employee[keys]);
    }


    Output:-
    name: Ram
    salary: 200000
    address: kathmandu
    age: 26

    In above case "keys" is simply a variable. When we do employee[keys], consecutive key in stored in the "keys" variable and output is displayed.

    • Using for-in loop in array.

      • let names = ["Ram", "Shyam", "Hari", 12, "Sita"];
      for(name in names){
          console.log(name);
      }


      Output:-
      0
      1
      2
      3
      4

    This is because behind the scenes, a JavaScript array is just an object where each element is associated with a key (the index of the element). Similar to the for…in loop on a JavaScript object, when called on an array, the for…in loop grabs these “invisible” keys (indexes) instead of the array values. This is why you should not use a for…in loop to iterate over arrays. It simply doesn’t work the way you’d expect.
