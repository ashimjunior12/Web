Data Types

var age = 26;
var last-name = "reings";
var cars = ["Saab", "Volvo", "BMW"];
var x = {firstName:"John", lastName:"Doe"};

In the above code, age is integer, last-name is a string, cars is an array & x is an object.

JavaScript evaluates expressions from left to right.

So, while JavaScript uses the second expression for the data type to work with, if you have multiple sequences, then the results are different.

var x = "16" + "Blue";         //Output is 16Blue.
var x = 16 + 4 + "Blue";      //Output is 20Blue.
var x = "Blue" + 16 + 4;     //Output is Blue164.

JavaScript has DYNAMIC TYPES, meaning that the same variable can be used more than one type.

var x;            //x is undefined
var x = 9;         //Now x is a number
var x = "Ram"     //Now x is a string
