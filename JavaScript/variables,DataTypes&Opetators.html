Variables, Data Types & Operators

var keyword:- It is used to declare a variable in JavaScript. Example:- var a. In this case, we have created a variable named 'a' using var keyword.


- Two types of operators in JavaScript:-
  1. Unary Operator:- Has only one operand. Example, a=-1;, In this case - is operator and 1 is operand.

  2. Binary Operator:- Has two operands. Example, a=1+2, a=b+c.

- Some Arthmetic operators:

  1. Exponentiation Operator(**):- It is used to raise a number to a certain power. Example:- 'base ** exponent', ie, 2**3. In this case 2 is raised 3 times and the final output is 8.

  2. Increment/Decrement Operator(++/--):- In JavaScript, the increment (++) and decrement (--) operators are used to increase or decrease the value of a variable by 1, respectively. Example:-

    var num1=6;
    num1++;
    console.log(num1);


    The output is 7 cause the value of num1 is increased by 1. The value is decreased by 1 while using decrement operator.

    Now let's observe a slight difference between num1++ and ++num1.


    num1 = 6;
    console.log(++num1);


    In this case, we will get the output as expected.i.e. 7.
    However, in this case,

    num1=6;
    console.log(num1++);


    The output is 6. The reason is simple. First the number is printed (i.e, initial value,6) after that the number is increased by 1 and then, num1 is updated as num1=7;

-More on Binary Operators:-

  1. Arithmetic Operators:- Use to perform mathmetical operations. Example:- '+', '-', '/', '*', '%'.

  2. Assignment Operators:- Assigns values from right side operands to the left side operand.

    Example:- Addition Assignment (+=).

    Addition Assignment adds the right operand to the left operand and assigns the value to the left operand.

    var y = 10;
    y += 3;     //Now the value of y is 13.

  3. Comparison Operators or Relational Operators:- Used in programming to compare values and determine the relationship between them. These operators return a Boolean value (true or false) based on whether the comparison is true or false.

    Example:- < , >, <=, >=, ==, !=

  4. Binary Logical Operators (&&, ||, !)
  5. Binary Bitwise Operators (&, ^, |)
