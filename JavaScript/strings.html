Strings in JavaScript

Single quotes string:-

var name = 'ram';    //we can use "" inside of it

Double Quotes String:-

var name = "hari";    //We can use single quotes inside of it

Escape Sequences:-

var car_name = "tesla\naudi"    //audi is printed in next line

String Concatenation

var name="ram";
var age = "18";
console.log(name + " is "+ age + " years old");

Output:- ram is 18 years old

One string makes everything string

var a = "1";    //a is a String
var b = 2;     //b is a number
console.log(a+b);

Output:- 12, which is a string

Concatenation is only when we have + symbol

var a = "3";    //string
var b = 2;    //integer
console.log(a*b);

Output:- 6, which is integer

Other ways to concatenate

  1. Using assignment operator:-

    var names = "ram";
    names += "shyam";
    names+= "hari";
    console.log(names);

    Output:- ramshyamhari

  2. Using Built-in concat method:-

    var name = "".concat("hello", " world", "1");
    console.log(name);

    Output:- hello, world!

  3. Using backtick(``):-

    var name = "ram";
    var age = 18;
    console.log(`${name} is ${age} years old.`);

    Output:- ram is 18 years old.

Way of finding the length of String:-

.length method is used to determine the length of the string.

var name = "Ram";
console.log(name.length);

Output:- 3
