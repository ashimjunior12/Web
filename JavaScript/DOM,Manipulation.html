How to manipulate the DOM

DOM:- DOM stands for Document Model Object. It can be understood as a tree of nodes which has its own methods and properties. Image below shows how a DOM tree looks like.

Here we have document object. To perform any form of DOM manipulation, first we have to access the document object. Every elements in HTML document is represented as an object.

After the document, we have html root element, which is the child element of document object.

Next we have head and body which are children of html element and are siblings of each other.

Under head element, we have title which is a child of head element and parent to text node - "My Title".

Similarly, we have a & h1 element which are children of body element and parent to next text nodes.


4 major ways of selecting DOM elements

  1. Using getElementById() method

    Since id's are unique, it is easier to grab the element using this method.

    <p id="para"> This is a paragraph </p>
    <script>
      const para = document.getElementById("para");
      console.log(para);         //<p id="para"> This is a paragraph </p>
    </script>

    Id is case sensative, para and Para are totally different things.

  2. Using getElementsByClassName()
    Since mulitple elements can have same class name, this method returs a collection of all matching elements.

    <p class="content"> This is a paragraph</p>
    <p class="content"> This is another paragraph</p>
    <h1 class="content">Hello world</h1>

    <script>
       const content = document.getElementsByClassName("content")[2];
       console.log(content);      //<h1 class="content">Hello world</h1>;
    </script>


  3. Using getElementsByTagName()
    Similar to the above method, this also returns the collection of matching HTML elements.

    <p class="content"> This is a paragraph</p>
    <p class="content"> This is another paragraph</p>
    <p class="content"> This is a third paragraph</p>

    <script>
        const content = document.getElementsByTagName("p");
        console.log(content[1]);     //<p class="content"> This is another paragraph</p>
    </script>


  4. Using .querySelector()
    This method returs the first match from the CSS selector assigned to it.

    <p class="content"> This is a paragraph</p>
    <p class="content"> This is another paragraph</p>
    <p class="content"> This is a third paragraph</p>

    <script>
        const content = document.querySelector(".content");
        console.log(content);      //<p class="content"> This is a paragraph</p>
    </script>

    To select all the matching elements, we use another method. i.e.

    .querySelectorAll()
    This method returns collection of all the matching elements(usually a node list) from the HTML document.

    <p class="content"> This is a paragraph</p>
    <p class="content"> This is another paragraph</p>
    <p class="content"> This is a third paragraph</p>

    <script>
        const content = document.querySelectorAll(".content")[0];
        console.log(content);      //<p class="content"> This is a paragraph</p>
    </script>


Traverse over the Document

We can use the following ways to traverse over various elements.
  • parentNode
  • childrenNodes
  • firstElementChild
  • lastElementChild
  • previousElementSibling
  • nextElementSibling
.children():- returns a collection of all the child elements from its parent element.

.firstElementChild():- returns the first child element from its parent element.

.lastElementChild():- returns the last child element from its parent element.

.previousElementSibling():- returns the previous sibling element with respect to the selected element.

.nextElementSibling():- returns the next sibling element with respect to the selected element.

.parent():- returns the parent element of the selected element.


Inserting HTML element

.createElement('element_name') is used to create a new HTML element.

.innerHTML() property is used to add some text nodes insed of it.

parentElement.appendChild() method is used to append the newly created HTML element at the very last.

.insertBefore(newNode, existingNode):- is used to insert the newly created HTML element before the desire element.
parentElement.insertBefore(newNode, existingNode);

.removeChild() is used to remove the child element from its parent element.

.replaceChild(newNode, existingNode) is used to replace elements from parent element.

.classList.add('class_name') method is used to add class and add some styling in the desired element.

.classList.remove('class_name') method is used to remove class from desired element.

.classList.toggle('class_name'):-
If it has the class and you click the button, it REMOVES it. If it doesn't have the class and you click the button, it ADDS it.

.addEventListener():- This method listens events happening in HTML. This method takes 3 arguments.
First one is event:- "click", "mouseover", "mouseout", etc...
Second is:- function that is to be executed while event occurs.
Last is:- boolean value(optional)
