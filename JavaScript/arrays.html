Arrays

Array is an object that can hold multiple data at once.
Data inside an array can be integer, string, object, functions.
Arrays are mutable datatypes meaning their datas can be altered or changed.
  • Creation of array
    1. Using array literal([])

      Syntax:

      const array_name = [item1, item2, item3, ...., itemn];
      //It is a common practice to declare arrays with the const keyword.

      Example:-

      const names = ["Ram", "hari", 1, "shyam"];

      We can first declare array and then assign items to it.
      Example:-

      const cars = [];
      cars[0] = "BMW";
      cars[1] = "audi";
      cars[2] = "toyota"
      console.log(cars);

      Output:- ["BMW", "audi", "toyota"]

    2. Using array constructor new Array()

      Syntax:-

      const array_name = new Array(item1, item2, item3, ..., itemn);

      Example:-

      const names = new Array("ram", "shyam", "hari");
      console.log(names);

      Output:- ["ram", "shyam", "hari"]

    For simplicity, readability and execution speed, we use the array literal method.

  • Accessing Arrays:-
    We can access datas from arrays using index. Indexing starts from 0.

    const names = ["Ram", "Shyam", "Hari"];

    If I want to get the first data from "names" array then:-

    console.log(names[0]);    //Ram


  • Changing array element
    We can use array index to change elements inside of it.

    const names = ["Ram", "shyam", "hari"];
    names[0] = "sita";
    console.log(names);

    Output:- ["sita", "shyam", "hari"]   //The first element is changed to "sita".

  • Changing Last Element Of Array Using pop() & push() methods:-
    We can use pop() method to delete the last element from an array.

    const names = ["ram", "shyam", "hari"];
    names.pop();

    Output:- ["ram", "shyam"];   //The last element gets deleted

    We can use push() method to add element at the last of an array.

    const names = ["ram", "shyam", "hari"];
    names.push("sita");
    console.log(names);

    Output:- ["ram", "shyam", "hari", "sita"]    //"sita" is added at the last

  • Changing First Element Of array using shift() & unshift() method.
    We can use shift() method to delete first element of an array.

    const names = ["ram", "shyam", "hari"];
    names.shift();
    console.log(names);

    Output:- ["shyam", "hari"]    //"ram" is deleted

    We can use unshift() method to add element at the beginning of an array.

    const names = ["ram", "shyam", "hari"];
    names.unshift("sita");
    console.log(names);

    Output:- ["sita", "ram", "shyam", "hari"]    //"sita is added at index 0"

  • Array's method:-
    .length :- .length is used to find out the total no of elements in array.
    .sort:- .sort() method is used to sort array in ascending order.

    • Arranging array in descending order using .sort() and .reverse() method.

      const numbers = [12, 19, 14,10, 3, 2];
      console.log(numbers.sort().reverse());

      // The numbers are first sorted and then reversed which will give the output in descending order.



    Array elements can be deleted using the JavaScript operator delete. Using delete leaves undefined holes in the array. Use pop() or shift() instead.
    Example:-

    let names = ["ram", "shyam"];
    delete names[0];
