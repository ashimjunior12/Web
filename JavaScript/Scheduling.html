Scheduling

If we want our function to execute after a period of time, then we may schedule the calling of function
There are two methods of doing it.
  • setTimeout:- allows us to run a function after specified time and the function is executed only once

  • setInterval:- allows us to run a function after specified time and the function is executed continuously



    • setTimeout:-
    Syntax:-

    setTimeout(func|code, [delay], [arg1], [arg2], ....);

    func:- It takes refrence of function. If in case we call the function, then it throws an error. So, we have to pass the refrence of function.

    delay:- delay is measured in milliseconds. i.e. 1000 = 1sec

    argument:- arguments are sent to the function.

    function greet(user, greeting){
       console.log(`Hello ${user}, ${greeting}`);   //Hello Hari, Good Morning is printed after 5 seconds
    }
    setTimeout(greet, 5000, "Hari", "Good Morning");

    setTimeout is commonly used for displaying notifications after certain interval of time.


    setInterval:-
    syntax is similar to setTimeout. But this function is used to execute a function in a regular interval of time. It is commonly used while making animations, updating contents.

    function greet(user, greeting){
       console.log(`Hello ${user}, ${greeting}`);   //Hello Hari, Good Morning is printed after every 5 seconds
    }
    setInterval(greet, 5000, "Hari", "Good Morning");



setInterval fucntion returns us an interval id which we can use to stop the interval using clearTimeout(interval_id).
